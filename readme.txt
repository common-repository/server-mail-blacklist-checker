Server Mail Blacklist Checker checks if your website or server IP is listed in DNSBL (DNS-based Blackhole List) and RBL (Real-time Blackhole List) blacklists. 

== Description ==

When a blacklisted mail server attempts to send an email to a another mail server that is protected against spam, the target mail server refuses to accept the message. Or you server IP can be blocked and blacklisted by one of hundreds spam databases, that means that your customers will never get mail from you and would not be able to register on your site and confirm their email. Our plugin checks more than 100 different databases and sends you a notification if your site or mail server has been listed in one of databases. It can perform daily, weekly or monthly automated checks and send you email notifications. Even of your mail server has been listed our plugin can help you remove your site from any black list.

**Main features:**

* More than 100 databases
* Automated checks
* Semi-automated blacklist removal
* Daily, Weekly, MOnthly check

**Benefits**
Using Server Mail Blacklist Checker quickly after a suspicion of having a blacklisted server arises can save you a lot of time and prevent lots of troubles. With a single click you will get information from more than 100 blacklists on whether your server is blacklisted or not.

Plugin is using 3rd party service. Requests to API (Application Programming Interface) of SiteGuarding.com
API makes queries to all mail blacklist and checks your website if it's blacklisted or not. 
It sends your domain information with API requests to siteguarding.com server.
Link to the services: https://www.siteguarding.com/en/wordpress-security-services
Link to terms of use: https://www.siteguarding.com/docs/terms-of-use.pdf

== Installation ==

1. Upload all the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
